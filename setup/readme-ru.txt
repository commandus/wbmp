Назначение
----------
Позволяет преобразовывать изображения MS Windows BMP, GIF, JPEG,
ICO, WMF в формат WBMP, тип 0 (черно- белый, 1 бит, без компрессии) 
для приложений WAP. Также Можно сохранить изображение в текстовый
файл.
